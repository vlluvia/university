## google

1. 网址
> https://www.exploit-db.com/google-hacking-database

2. 基本语法

```shell 
1. site         ----搜索指定网站的内容

2. intitle        ----搜索网站标题中包含此关键字的网站

3. allintitle       ----搜索标题中包括多个关键字的网页

4. inurl        ----搜索网站url中包含此关键字的网站

5. allinurl        ----搜索网站url中包含多个关键字的网站

6. intext      ----搜索正文中包含关键字的网页

7. filetype       ----指定文件类型 

8. link       ----查询有链接到指定链接的网站

```

## shodan

1. 网址
> https://www.shodan.io
```shell 
hostname                —搜索指定的主机或域名，例如 hostname:baidu
port                    —搜索指定的端口或服务，例如 port:80
country                 —搜索指定的国家，例如 country:US
city                    —搜索指定的城市，例如 city:Chengdu
org                     —搜索指定的组织或公司，例如 org:"Google"
isp                     —搜索指定的ISP供应商，例如 isp:"China Telecom"
product                 —搜索指定的操作系统/软件/平台，例如 product:"Apache httpd"
version                 —搜索指定的软件版本，例如 version:"1.6.2"
geo                     —搜索指定的地理位置，参数为经纬度，例如 geo:"31.8639, 117.2808"
before/after            —搜索指定收录时间前后的数据，格式为dd-mm-yy，例如 before:"11-11-15"
net                     —搜索指定的IP地址或子网，例如 net:"210.45.240.0/24"
```
2. 命令行
```shell 
    git clone https://github.com/achillean/shodan-python.git
    python setup.py
    
    shodan init YOUR_API_KEY
    shodan count +"查询内容" 是统计条目数
    
    shodan download apche-data apche
    shodan parse --fields ip_str,port,org --separator , apache-data.json.gz
    
    shodan myip 看自己的IP
    shodan search --fields ip_str,port,org,hostnames apache
    
    shodan host +ip
    shodan host 211.0.0.1/24
    
    shodan stats apche
```

3. python

* 安装

* 库
```shell 
shodan.Shodan(key) ：初始化连接API
Shodan.count(query, facets=None)：返回查询结果数量
Shodan.host(ip, history=False)：返回一个IP的详细信息
Shodan.ports()：返回Shodan可查询的端口号
Shodan.protocols()：返回Shodan可查询的协议
Shodan.services()：返回Shodan可查询的服务
Shodan.queries(page=1, sort='timestamp', order='desc')：查询其他用户分享的查询规则
Shodan.scan(ips, force=False)：使用Shodan进行扫描，ips可以为字符或字典类型
Shodan.search(query, page=1, limit=None, offset=None, facets=None, minify=True)：查询Shodan数据
```
