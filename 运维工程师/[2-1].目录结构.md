
# 目录结构

```shell 
/bin                            命令、二进制文件存放目录
/boot                           系统引导程序+系统内核
/dev                            设备 光驱 硬盘
/etc                            存放系统或服务的配置文件
/home                           普通用户的家目录
/lib                            库文件存放目录
/lib64                          库文件存放目录(64位)
/lost+found                     硬盘、文件系统损坏 断电 临时存放文件
/media
/misc
/mnt                            临时挂载点
/opt                            第三方软件安装在这
/proc                           虚拟目录，显示内存中信息（进程 服务信息 内核）
/sbin                           超级命令，只有root用户能用 
/selinux                        selinux及配置文件存放位置 /etc/selinux
/srv
/sys                            虚拟目录，内存信息
/tmp                            临时文件存放位置，垃圾堆
/usr                            存放用户的程序
/var                            经常变化的文件 系统日志 相关服务日志
/swap 交换分区 ，内存不够，将硬盘中的空间充当内存
        内存 < 8G
        建议交换分区 = 内存大小的1.5倍
        内存 >= 8G
        建议交换分区大小 = 8G
/data 剩余空间大小
```
