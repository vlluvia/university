
# 系统特殊符号

```shell 
        ~                               home目录符号
        -                               两个目录之间快速切换($OLDPWD)
        ./                              当前路径
        ../                             上一级目录
        >                               标准输出重定向符号，直接覆盖
        2>                              错误输出重定向符号
        >>                              标准输出重定向符号，文件末尾追加信息
        2>>                             错误输出重定向符号
        <                               标准输入重定向
        <<                              标准输入追加重定向
        &&                              前一个命令执行成功之后执行后一个命令、与
        ||                              前一个命令执行失败之后执行后一个命令、或
        #                               注释
        ``                              将反引号内部的执行结果传给反引号外的命令
        $                               调取变量、命令的提示符、行尾标志、awk取列
        !                               取反或者排除、命令行表示最近命令、表示强制操作（vim）
        |                               管道符号，前一个命令执行结果交给后面命令处理
        ''                              输出信息不会改变
        ""                              和单引号一致，但是会对特殊信息过滤
        *                               模糊匹配
            find / -name "*.txt"
        {}                              生成序列信息（生成一行序列）
            exho {1..10}
            exho {01..100}
            exho {a..z}
            exho {A..Z}
            exho 1{A,Z}
            exho {1..10..2}
            exho {a..z}{01..03}
            exho A{,02}
            cp /opt/file{,.bak}     -- 快速备份

```
