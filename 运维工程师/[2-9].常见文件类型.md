
# 常见文件类型

```shell 
        -- 普通文件
        a. 存文本文件: text /etc/hosts                       -- 文本文件
        b. 二进制文件: binary /bin/ls                        -- 命令文件 
        c. 数据型文件: data /tmp/etc.tar.gz                  -- 数据文件 
        
        -- 链接文件（快捷方式）
        -- 块文件/字符文件（设备文件）
            块文件：  存储的设备文件 光驱设备文件 磁盘
            字符文件：持续输出的字符信息

            磁盘设备文件 /dev/sda                            -- 磁盘文件 
            随机设备文件 /dev/urandom 
            输出设备文件 /dev/zero                           -- 黑洞出口文件
            输入设备文件 /dev/null                           -- 黑洞入口文件

        -- socket文件

```
