

# 函数


## 函数


func main() {
    res := sum(5, 9)
}


* 一般函数格式
```go 
package main

import "fmt"

func sum(x int, y int) (ret int) {
    return x + y
}

func main() {
    res := sum(5, 9)
}
```
* 没有返回值
```go 
func sum(x int, y int) {
    fmt.Println("哈哈")
}
```

* 没有参数没有返回值
```go 
func sum(){
    fmt.Println("哈哈")
}
```

* 没有参数但有返回值
```go 
func sum() int {
    return 100
}

```
* 参数可以命名也可以不命名
```go 
func sum(x int, y int) (ret int) {
    ret := x + y
    return 
}
```

* 多个返回值
```go
func sum() (int, string){
    return 100, "哈哈"
}
```

* 参数缩写
```go 
func sum(x, y int) int{
    return x + y
}
```

* 可变长参数
```go 
func sum(x string, y ...int){
    fmt.Println(x)
    fmt.Println(y)
}
```
