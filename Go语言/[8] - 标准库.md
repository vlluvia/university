

# 标准库

* fmt

## fmt
* 输出数据
```go 
func Print()
func Println()
func Printf()
    %T  : 查看类型
    %d  : 十进制数
    %b  : 二进制数
    %o  : 八进制数
    %x  : 十六进制数
    %c  : 字符
    %s  : 字符串
    %p  : 指针
    %v  : 值
    %t  : bool
   
```

* 终端输入
```go 
var s string
fmt.Scan(&s)
fmt.Scanln(&s)      //扫描一行

var(
    name string 
    age int
)
fmt.Scan("%s %d", &name, &age)
```
