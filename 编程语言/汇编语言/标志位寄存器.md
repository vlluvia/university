# 标志位
* CF
* ZF
* PF
* SF
* OF
* DF
## CF

进位寄存器，和运算指令有关

* CY == 真
* NC == 假


## ZF

结果是否为0或者判断是否相等

* ZR == zero
* NZ == not zero


## PF

判断结果里1的个数的奇偶性  

0000 0011 偶  
0000 1110 奇


* PE == even 偶数
* PO == odd  奇数


## SF

符号，正或者负

* NG == negative 负数
* PL == positive 正数



## OF

运算结果是否溢出

* OV == overflow 溢出
* NV == not overflow 没溢出


## DF


* DN == down 向上
* UP == up  向下
