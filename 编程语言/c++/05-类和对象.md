## 静态成员
* 静态成员变量
``` 
所有对象共享同一份数据
在编译阶段分配内存
类内声明，类外初始化

class Person{
    public:
        static int a;
};

int Person::a = 100;
void test(){
    Person p;
    cout << p.a <<endl; // 100
    
    Person p2;
    p2.a = 200;
    cout << p.a <<endl; // 200
    
    
}

```
* 静态成员函数
``` 
所有对象共享同一个函数
静态成员函数只能访问静态成员变量 


class Person{
    public:
        static void func(){
        
        }
};

Person::func();

```
