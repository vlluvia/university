# 函数提高

``` 
int func(int a, int b = 20, int c = 30){
    return a + b + c;
}

cout << func(10) <<endl;

```

## 函数声明

``` 
// 声明和实现只能有一个默认参数
int func(int a = 10, int b = 10);

int func(int a = 20, int b = 20){
    return a + b;
}
```

## 函数占位参数
``` 
void func(int a, int){
    cout << a << endl;
}

func(10, 10)
```

## 函数重载
> 函数名可以相同，
``` 
void func(int a){}

void func(string a){}
```
